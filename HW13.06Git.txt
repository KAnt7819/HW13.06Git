Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?


Как получить на локальный компьютер изменения с github.com

Какую информацию нам дают команда :
git status
git log
git remote -v 

Для того чтобы скопировать репозиторий из гитхаб , предварительно нам нужно добавить в свой аккаунт 
в гитхаб ssh Key ключ из гитбаш.
Затем, чтобы склонировать репозиторий, нажимаем <>code в гитхаб, выбираем протокол ssh и нажимаем copy, в гитбаш возращаемся в тот репозиторий куда нам нужно скачать. Далее, используем команду "git clone" и вставляем ссылку из гитхаб.

Команда "git status" - покажет нам состояние файлов в репозитории гит 
Untracked files + git add <file> + имя файла

"git log" - нам показывает все комментарии, которые были сделаны в репозитории и подписаны авторами
(Время и Дата и уникальные хешкоды для коммита)


С помошью  "git remote -v" мы убедимся, что наш локальный репозиторий привязан к гитхаб. Команда дает нам 
"origin push" - означает отправлять в гитхаб
"origin fetch" - означает получать их гитхаб
